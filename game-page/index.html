DOOM

Publishing Info

  • Published by: id Software, Inc.
  • Developed by: id Software, Inc.
  • Released: Dec, 1993

Description

DOOM is another first-person 3D shooter: full-on action seen from the space marine's perspective.
The basic objective in each level is simply to reach the exit. Since dozens of enemies stand in the way, the only way to get there is through killing them.
Switches and buttons must be pressed to advance at certain points and often color-coded locked doors will block the way - matching keycards or skull keys must be found to pass.

Game Categories

  • Genre: Action
  • Perspective: 1st-person
  • Setting: Sci-fi / futuristic
  • Gameplay: Shooter
